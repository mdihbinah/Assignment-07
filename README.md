# 🤝 KeenKeeper
Keep Your Friendships Alive

# 📝 Description
KeenKeeper is a modern web application designed to help users maintain and manage meaningful relationships. In a busy world, it’s easy to let time slip by without reaching out to loved ones. This app allows users to track interactions, monitor communication frequency, and stay connected with friends through an intuitive interface.

# ⭐ Key Features
## ① 👫 Friend Management
- View all friends in a responsive grid layout  
- Each friend has detailed profile information  
- Status indicators (Overdue, Almost Due, On-track)  

## ② 📜 Timeline Tracking
- Displays call, text, and video history  
- Includes filter functionality (Call / Text / Video)  


## ③ 📊 Analytics Dashboard
- Visual pie chart of interactions  
- Helps understand communication habits  
- Clean and interactive UI
